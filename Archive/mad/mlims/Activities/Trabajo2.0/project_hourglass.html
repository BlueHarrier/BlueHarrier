Hourglass Framework

Surgido en 2018, este marco de trabajo dota al motor Game Maker de la capacidad de mover elementos 3D manteniendo la esencia y patrones de programación y diseño originales. El marco consta de dos partes:

La librería

Internamente, Game Maker necesita una forma de ejecutar los elementos básicos que componen el motor. Para ello, la librería consta de unos shaders básicos de PBR, un sistema de control de capas 3D y funciones y clases para controlar objetos tridimensionales. Además del sistema gráfico, también constará de un motor de colisiones para este nuevo espacio.

El editor

Lamentablemente, Game Maker Studio no dispone de soporte nativo para integrar nuevos elementos de este estilo. Para ello es necesario un editor externo encargado de cargar, parsear y comprimir los archivos útiles en un solo .dat del cual el juego cargará el conjunto de assets preparados, incluyendo modelos, entornos, salas y texturas.

Este editor también posee una variación del Harrier's Shader Graph, capacitado y optimizado para manejar iluminación 3D, armaduras y blendshapes.
