Harrier's Shader Graph

Este proyecto comenzó como una escisión del Hourglass Framework, dado que un editor de shaders visual era necesario para dicho proyecto, aunque también puede aplicarse a Game Maker general, pues actualmente no existe ninguno.

Se trata de un proyecto Open Source, que se encontrará disponible a través de GitHub para compilar y modificar, y se podrá descargar a través del marketplace oficial de Game Maker.

Hoy en día es difícil encontrar un solo motor de juegos que no disponga de un editor visual integrado de shaders por nodos. Desafortunadamente, Game Maker es uno de estos escasos editores. Por ello, Harrier's Shader Graph ofrece esta posibilidad, permitiendo la creación de shaders utilizando bloques para parsearlos a GLSL ES (compatible) ó HLSL y GLSL (nativo) para que Game Maker los compile al ejecutar el juego. También permitirá la importación de librerías de funciones en cualquier lenguaje que serán introducidas dentro del editor como bloques con entradas y salidas, y serán transferidas al shader final.
